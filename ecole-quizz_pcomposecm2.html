🎯 Objectif

Avec avoir : le participe passé ne s’accorde généralement pas.
Exemple : j’ai mangé, elle a vu, nous avons pris.
Avec être : le participe passé s’accorde avec le sujet.
Exemple : il est parti, elle est partie, ils sont partis, elles sont parties.
⭐ Verbes essentiels :
être → j’ai été
avoir → j’ai eu
